# A股动量因子研究

## 项目简介
本项目研究A股市场中短周期动量因子的横截面预测能力。基于2020–2025年的样本数据，通过标准的量化因子研究流程，发现20日动量因子在A股市场中并未呈现传统的正向动量效应，而是表现出显著的**短期反转特征**。

---

## 方法

- **因子构建**
  - 构建20日动量因子
  - 未来收益率
- **因子预处理**
  - 横截面去极值（5%–95%）
  - 横截面标准化（Z-score）
  - 行业与市值中性化（OLS回归）
  - 中性化后再次标准化

- **因子评估方法**
  - Rank IC 分析（Spearman相关系数）
  - 分层回测（10分组等权组合）
  - 多空组合（Group 0 - Group 9）
  - 横截面回归（OLS）

---

## 核心结果

### Rank IC 表现
- **IC均值**：-0.0556  
- **IC标准差**：0.0985  
- **ICIR**：-0.5640  
- **IC正比例**：29.20%

结果表明，该因子整体呈现**负向预测能力**，说明高动量股票未来收益较差，体现出短期反转效应。

### 分层回测结果（多空组合）
- **年化收益率**：15.43%
- **年化波动率**：13.25%
- **Sharpe比率**：1.15
- **最大回撤**：-11.45%

分层结果显示，低因子组（Group 0）显著优于高因子组（Group 9），收益不具备单调递增特征，进一步验证反转效应。

### 横截面回归结果
- **Beta均值**：-0.00396  
- **Beta > 0 比例**：37.70%  
- **t值均值**：-2.187  
- **|t| > 2 比例**：71.27%  
- **p < 0.05 比例**：72.05%

回归结果显示因子系数显著为负，统计显著性较强，进一步支持短期反转结论。

---

## 研究结论

综合 Rank IC、分层回测及横截面回归结果，可以得出以下结论：

> **A股市场中，20日动量因子呈现显著的短期反转效应，而非传统的正向动量效应。**

该因子更适合作为反向因子（Reversal Factor）进行应用，即：
- 做多低动量股票
- 做空高动量股票

## 回测设定

- **股票池**：A股全市场
- **样本区间**：2020 – 2025
- **调仓周期**：20个交易日
- **分组方式**：10分组
- **权重方式**：等权
- **中性化处理**：行业 + 市值
